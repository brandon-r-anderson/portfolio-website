# Portfolio Website — Data Analytics Projects

This repository contains a personal portfolio website showcasing data analytics projects, technical skills, and professional experience. The site is designed to present applied analytics work through structured case studies, dashboards, and data-driven insights.

## Overview

The portfolio highlights end-to-end analytical workflows, including data cleaning, exploratory analysis, and visualization using tools such as Excel, Python, SQL, and Power BI. Each project is structured to demonstrate problem-solving, data transformation, and insight generation from real-world datasets.

## Projects Featured

### Game Graphics Performance Analysis: The Cost of Visual Fidelity
- Analyzed 72 benchmark runs across 4 AAA games to evaluate FPS stability, CPU/GPU utilization, and thermal performance
- Processed and structured HWINFO telemetry data for performance analysis
- Built analytical models and dashboards to evaluate tradeoffs between graphical fidelity and system performance

### Movie Performance & ROI Analysis: From Budget to Blockbuster
- Cleaned and transformed a raw Kaggle dataset using Excel Power Query
- Analyzed relationships between budget, revenue, popularity, and return on investment
- Built an interactive Excel dashboard using pivot tables and pivot charts to visualize trends in film performance
